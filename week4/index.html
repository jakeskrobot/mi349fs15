wut-the-web

A tutorial website dedicated to the basic/intermediate fundamentals of web-design.

About

wut-the-web will focus on teaching aspiring web-designers the fundamentals of web-design and how to code web pages from text-editing software. By utilizing strong informational graphics and well-typed descriptions, this site should, in theory, allow a user who has no prior knowledge of web-development to create a basic web page and also add more advanced options to their page. The site may also allow users to communicate amongst other web development aficionados and experts in a forum type setting.
